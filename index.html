The Crystal Collector

You will be given a RANDOM number at the start of each round

There are FOUR Crystals below. Clicking on a crystal will add a specific number of points to your total score

Win by matching YOUR totl score to the random number. If you go OVER the random number, you lose!

The VALUE of each crystal is HIDDEN until you click on it

Each time the game resets, the value of the crystal will CHANGE

Make sense? If not, start the game and see if you can figure it out by doing!
